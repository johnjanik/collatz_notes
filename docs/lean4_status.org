**** Proposed decomposition (Diophantine Repeller strategy)

The following decomposition would refine the sorry into smaller,
more targeted claims:

1. =hensel_attrition= (PROVED, HenselAttrition.lean):
   d consecutive v₂=1 steps iff 2^{d+1} ∣ (x+1).
   Attrition rate exactly 2^{-d}. No Baker needed.

2. =baker_cell_separation= (PROVABLE from Baker chain):
   Dangerous cells on (ℤ/3^kℤ)² are separated by distance
   > C/q^κ (Baker-Feldman effective bound).

3. =finite_residence_bound= (OPEN — the irreducible gap):
   No trajectory sustains more than M consecutive steps in
   the dangerous set. Requires showing trajectories cannot
   regenerate trailing 1-bits fast enough — a mixing property
   of the Collatz map on the (2,3)-solenoid.

4. =k_bound_from_repeller= (PROVABLE from finite_residence):
   Sliding window argument: safe steps in any window of size
   W > M outweigh dangerous steps, establishing 3ν₃ ≤ t + K.

The chain: hensel_attrition + baker_cell_separation
→ [finite_residence_bound: sorry] → k_bound_from_repeller
→ nu3_linear_bound → reaches_one → collatz_conjecture.
